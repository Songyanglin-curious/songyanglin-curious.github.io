发表于 2025-02-05 更新于 2025-02-24 分类于 教程 ， 快捷键 本文字数： 1.5k 阅读时长 ≈ 3 分钟

一、核心快捷键分类

1. 文件与项目操作

  • Ctrl + N ：新建文件
  • Ctrl + Shift + N ：新建项目
  • Ctrl + O ：打开文件
  • Ctrl + S ：保存当前文件
  • Ctrl + Shift + S ：保存所有文件
  • Ctrl + W ：关闭当前标签页
  • Ctrl + F4 ：关闭当前文档
  • Ctrl + Tab ：切换打开的标签页

2. 代码编辑

  • Ctrl + Space ：触发智能提示（代码补全）
  • Ctrl + K → Ctrl + C ：注释选中行
  • Ctrl + K → Ctrl + U ：取消注释
  • Ctrl + D ：复制当前行到下一行
  • Ctrl + X ：剪切当前行（未选中时）
  • Alt + ↑/↓ ：向上/向下移动当前行或代码块
  • Ctrl + R → Ctrl + R ：重命名符号（变量、方法等，自动更新所有引用）
  • Ctrl + . ：快速操作和重构（生成代码、修复建议等）

3. 代码导航

  • F12 ：跳转到定义（方法、类定义处）
  • Ctrl + - ：后退到上一个光标位置
  • Ctrl + Shift + - ：前进到下一个光标位置
  • Ctrl + T ：全局搜索符号（类、方法、文件）
  • Ctrl + F ：当前文件内搜索
  • Ctrl + Shift + F ：全局搜索（跨解决方案）

4. 调试与运行

  • F5 ：启动调试（继续运行）
  • Shift + F5 ：停止调试
  • F9 ：切换断点
  • F10 ：逐过程执行（Step Over）
  • F11 ：逐语句执行（Step Into）
  • Shift + F11 ：跳出当前方法（Step Out）
  • Ctrl + Alt + P ：附加到进程（Attach to Process）

5. 窗口与布局管理

  • Ctrl + Alt + L ：打开解决方案资源管理器
  • Ctrl + Alt + O ：打开输出窗口
  • Ctrl + Alt + E ：打开错误列表
  • **Ctrl + **, Ctrl + E ：打开命令窗口（输入命令）
  • Shift + Alt + Enter ：切换全屏模式

6. 代码格式化

  • Ctrl + K → Ctrl + D ：格式化整个文档
  • Ctrl + K → Ctrl + F ：格式化选中代码块

二、常见工作流组合

1. 新建项目并快速编码

场景 ：创建新项目并编写核心代码

  1. Ctrl + Shift + N → 新建项目（选择模板如 C# 控制台应用）
  2. Ctrl + Shift + A → 添加新项（如类、接口）
  3. Ctrl + Space → 触发智能提示补全代码
  4. Ctrl + K → Ctrl + D → 格式化代码

2. 调试与修复 Bug

场景 ：定位并修复运行时错误

  1. F9 → 在可疑代码行设置断点
  2. F5 → 启动调试，程序运行到断点暂停
  3. F10/F11 → 逐行执行，观察变量值（悬停变量名或使用 监视窗口 ）
  4. Shift + F5 → 停止调试，修改代码后重新运行

3. 代码重构

场景 ：优化代码结构或重命名

  1. Ctrl + R → Ctrl + R → 输入新名称，重命名变量/方法
  2. 选中代码块 → Ctrl + . → 提取方法（Extract Method）
  3. Ctrl + K → Ctrl + S → 包裹代码块（如 try-catch）

4. 多文件协作

场景 ：同时编辑多个文件并导航

  1. Ctrl + T → 搜索符号（如类名 MyClass）
  2. F12 → 跳转到定义
  3. Ctrl + - → 返回原文件
  4. **Ctrl + ** → 分屏编辑（左右布局）

5. 版本控制（Git）

场景 ：提交代码或解决冲突

  1. Ctrl + Alt + S → 打开团队资源管理器（Team Explorer）
  2. 输入提交信息 → Ctrl + Enter → 提交更改
  3. Ctrl + Shift + G → 查看文件差异

6. 单元测试

场景 ：运行并验证测试用例

  1. Ctrl + R → T → 运行当前测试方法
  2. Ctrl + R → A → 运行所有测试
  3. Ctrl + R → D → 调试当前测试

三、高级技巧

  1. 自定义快捷键 ：
  • 通过 工具 > 选项 > 环境 > 键盘 修改快捷键绑定。
  • 例如，绑定 Ctrl + E, Ctrl + W 为自动换行。
  1. 代码片段（Snippets） ：
  • 输入 ctor → Tab → 自动生成构造函数。
  • 输入 prop → Tab → 生成属性模板。
  1. 快速访问工具栏 ：
  • 按 Ctrl + 1 聚焦到代码编辑器，Ctrl + 2 聚焦到解决方案资源管理器。
