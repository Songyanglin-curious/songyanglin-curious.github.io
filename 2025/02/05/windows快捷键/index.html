windows快捷键

发表于 2025-02-05 更新于 2025-02-24 分类于 教程 ， 快捷键 本文字数： 2.4k 阅读时长 ≈ 4 分钟

1. 文件管理 & 整理

场景 ：批量整理文件、创建新项目文件夹

  • 操作流程 ：
  1. Win + E → 打开文件资源管理器
  2. Ctrl + Shift + N → 新建文件夹
  3. F2 → 重命名文件夹
  4. Ctrl + C → 复制文件夹路径
  5. Win + V → 打开剪贴板历史（粘贴路径到其他应用）

2. 多任务办公

场景 ：同时处理文档、浏览器、聊天工具

  • 操作流程 ：
  1. Win + Ctrl + D → 新建虚拟桌面（隔离工作区）
  2. Win + Tab → 查看所有窗口和虚拟桌面
  3. Alt + Tab → 快速切换应用
  4. Win + → → 将浏览器窗口贴靠到右侧（分屏操作）
  5. Win + D → 快速返回桌面查找文件

3. 文档编辑 & 格式调整

场景 ：高效撰写报告或邮件

  • 操作流程 ：
  1. Ctrl + N → 新建文档
  2. 输入内容后，Ctrl + S → 快速保存
  3. Ctrl + F → 查找关键词并修改
  4. Ctrl + B / I / U → 加粗、斜体、下划线调整格式
  5. Ctrl + P → 打印文档
  6. Alt + F4 → 关闭文档

4. 浏览器高效搜索

场景 ：快速查找资料并整理

  • 操作流程 ：
  1. Ctrl + T → 新建标签页
  2. Ctrl + L → 跳转到地址栏输入关键词
  3. Enter → 搜索后，Ctrl + F → 在页面内定位关键词
  4. Ctrl + C → 复制关键内容
  5. Ctrl + W → 关闭标签页
  6. Ctrl + Shift + T → 恢复误关闭的标签页

5. 截图 & 快速分享

场景 ：截取屏幕内容并发送

  • 操作流程 ：
  1. Win + Shift + S → 区域截图（自动保存到剪贴板）
  2. Win + V → 打开剪贴板历史 → 选择截图
  3. Ctrl + V → 粘贴到微信/邮件/文档
    （或直接 Win + E → 打开文件资源管理器保存截图）

6. 系统故障应急

场景 ：程序卡死或黑屏

  • 操作流程 ：
  1. Ctrl + Shift + Esc → 直接打开任务管理器
  2. 选择无响应的程序 → Del → 结束任务
  3. 若黑屏：Win + Ctrl + Shift + B → 重置显卡驱动

7. 会议 & 演示模式

场景 ：快速切换投影或多屏幕

  • 操作流程 ：
  1. Win + P → 切换投影模式（扩展/仅第二屏幕）
  2. Win + L → 临时锁定屏幕（离开座位时）
  3. Win + . → 快速插入表情符号（用于聊天工具）

8. 代码或文本批量处理

场景 ：编程或长文本编辑

  • 操作流程 ：
  1. Ctrl + ←/→ → 按单词移动光标
  2. Ctrl + Backspace → 删除整词
  3. Shift + ↑/↓ → 按行选择文本
  4. Ctrl + D → 多行选中相同内容（支持 VS Code 等编辑器）

9. 快速启动应用

场景 ：无需鼠标打开常用工具

  • 操作流程 ：
  1. Win + R → 打开“运行”窗口
  2. 输入命令（如 notepad、calc、cmd）→ Enter
  3. Win + 数字键 → 打开任务栏固定应用（如 Win+1 打开第一个图标应用）

10. 批量文件重命名

场景 ：整理照片或文档

  • 操作流程 ：
  1. Ctrl + A → 全选文件
  2. F2 → 输入新名称（如 旅行照片(1).jpg）
  3. Enter → 系统自动批量编号

通用快捷键

  • Ctrl + C ：复制选中内容
  • Ctrl + X ：剪切选中内容
  • Ctrl + V ：粘贴内容
  • Ctrl + Z ：撤销上一步操作
  • Ctrl + Y ：恢复撤销的操作
  • Ctrl + A ：全选当前页面/文档内容
  • Ctrl + S ：保存当前文件
  • Ctrl + P ：打开打印窗口
  • Ctrl + F ：打开查找功能（搜索当前内容）
  • Ctrl + N ：新建窗口/文档
  • Alt + Tab ：切换已打开的应用程序
  • Alt + F4 ：关闭当前窗口或程序
  • Win（Windows 徽标键） ：打开/关闭开始菜单

窗口与桌面管理

  • Win + D ：显示/隐藏桌面（快速最小化所有窗口）
  • Win + E ：打开文件资源管理器
  • Win + L ：锁定电脑（快速切换用户或进入锁屏）
  • Win + M ：最小化所有窗口
  • Win + Shift + M ：还原最小化的窗口
  • Win + ←/→/↑/↓ ：将窗口贴靠到屏幕左/右/上/下半部分
  • Win + Tab ：打开任务视图（查看虚拟桌面和近期活动）
  • Win + Ctrl + D ：创建新虚拟桌面
  • Win + Ctrl + ←/→ ：切换虚拟桌面
  • Win + + ：打开放大镜并放大屏幕
  • Win + Esc ：关闭放大镜

文件资源管理器

  • Ctrl + Shift + N ：新建文件夹
  • Alt + Enter ：查看选中文件/文件夹的属性
  • F2 ：重命名选中文件/文件夹
  • F5 ：刷新当前窗口
  • Alt + ←/→ ：后退/前进到上一个文件夹
  • Alt + D ：定位到地址栏（可直接输入路径）
  • Ctrl + Shift + Esc ：直接打开任务管理器

浏览器快捷键（适用于 Chrome/Edge 等）

  • Ctrl + T ：新建标签页
  • Ctrl + W ：关闭当前标签页
  • Ctrl + Shift + T ：恢复最近关闭的标签页
  • Ctrl + Tab ：切换到下一个标签页
  • Ctrl + Shift + Tab ：切换到上一个标签页
  • Ctrl + H ：打开历史记录
  • Ctrl + J ：打开下载列表
  • Ctrl + R / F5 ：刷新页面
  • Ctrl + F ：在页面内搜索关键词

文本编辑相关

  • Ctrl + B ：加粗文本（支持部分编辑器）
  • Ctrl + I ：斜体文本（支持部分编辑器）
  • Ctrl + U ：添加下划线（支持部分编辑器）
  • Ctrl + ←/→ ：按单词移动光标
  • Ctrl + Backspace ：删除前一个单词
  • Shift + ←/→/↑/↓ ：按字符或行选择文本

截图与录屏

  • Win + Shift + S ：打开截图工具（区域截图）
  • Win + Alt + R ：开始/停止屏幕录制（需开启 Xbox Game Bar）
  • PrtScn ：截取全屏并保存到剪贴板
  • Win + PrtScn ：截取全屏并自动保存到“图片”文件夹

高级操作

  • Win + R ：打开“运行”对话框（输入命令如 cmd、msconfig 等）
  • Win + X ：打开快捷菜单（类似右键开始按钮）
  • Win + I ：打开系统设置
  • Ctrl + Shift + Esc ：直接打开任务管理器（跳过 Ctrl+Alt+Del）
  • Win + Ctrl + Shift + B ：重置显卡驱动（黑屏时尝试恢复）

其他实用快捷键

  • Win + V ：打开剪贴板历史记录（需在设置中启用）
  • Win + .（句号） ：打开表情符号面板
  • Win + P ：切换投影模式（多显示器设置）
  • Alt + F8 ：在登录界面显示密码（需提前输入）
